1- Pour exécuter ce programme, les classes suivantes sont nécessaires : GraphA1NN, LabelledPoint, PointSet et UndirectedGraph.
2- Les fichiers knn.txt, ainsi que les fichiers de données (dataset) et de requêtes (query), doivent être placés dans le même dossier que GraphA1NN.java.
3- Pour lancer le programme, il est nécessaire de fournir les arguments suivants au programme principal (via la méthode : public static void main(String[] args)) :
▪ la valeur de k
▪ la valeur de s
▪ le nom du fichier de données
▪ le nom du fichier de point de requête
Par exemple, la commande pour exécuter cette classe serait :
java GraphA1NN 3 10 siftsmall_base.fvecs siftsmall_query.fvecs
